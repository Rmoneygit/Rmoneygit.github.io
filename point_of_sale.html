Point of Sale Application

This program is a generic point-of-sale application like Amazon, Wish, etc. After creating an account and logging in, users can browse the wide selection of products and add them to their shopping cart. Once they're ready, users can checkout and purchase items and get a mock receipt. Admins can upload images and create new products, as well as manage, add, and remove accounts.

I created this program in a team of 4 throughout the course of my Database & Applications class which I took in college.

Technologies used: Javscript, HTML, CSS, PHP, MySQL Workbench, Apache web server
