Self-Organizing Map

This program is a self-organizing map which runs on randomized training vectors as input data. It is an example of unsupervised machine learning. The goal of the program is to build a function that takes a 3-tuple as input, and associates it to 1 of 3 classes, represented by the colors red, green, and blue. The program runs in two phases. First, trains itself for 40 epochs on random groups of 120 training vectors. Then, the program allows you to input any vector you want, or to check a group of test vectors. Doing this allows you to verify if the map is classfying vectors correctly.

I created this program for an assignment in my Algorithm Engineering class which I took in college.

Technologies used: Javascript, HTML, CSS
